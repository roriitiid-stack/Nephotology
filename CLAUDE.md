# Nephotology — Claude Instructions

## GitHub Push Policy

This repo (`roriitiid-stack/Nephotology`) represents the **last-known-good checkpoint** of the firmware.

**Only push to GitHub when the user explicitly confirms that a build has been flashed and tested working on the actual ESP32-S3 hardware.**

- Do **not** push automatically on every local change.
- Always ask the user before pushing.
- A push means: "this build is confirmed working on hardware." Nothing else warrants a push.
